NotebookELF Packager

Build a full offline notebook .elf ZIP.

This applies the same browser-side packaging technique to the NotebookELF pipeline: corpus, manifest, source notes, runtime settings, and optional bundled model.

Notebook payload

Edit the seed passages, then package a portable notebook. The generated archive includes a working local viewer that loads the bundled model and answers from the notebook corpus.

corpusnotebook/corpus.json with passages generated from the seed text.
viewerA compact NotebookELF viewer that cites passage IDs and can use bundled SmolLM2.
runtimewllama worker + WASM copied from this site.
modelOptional full offline SmolLM2 GGUF bundle.
This is the export pipeline shape. The live NotebookELF app can adopt this writer later so user-created notebooks package the same way.
