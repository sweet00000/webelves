ELF Packager

Build a full offline Codelves .elf ZIP.

This builder runs in your browser. It downloads SmolLM2 from Hugging Face, adds the wllama runtime and a Codelves example project, then streams a ZIP to your disk. No server-side packing.

Package contents

The result is a local-first archive. Unzip it, serve the folder with any tiny static server, and open index.html.

modelmodel/smollm2.gguf from HuggingFaceTB/SmolLM2-360M-Instruct-GGUF.
runtimewllama worker + WASM files copied from this site so the package is not dependent on Cloudflare.
projectA small Codelves example workspace: Python, README, and a browser demo file.
settingsManifest + model settings: GGUF, Q4_K_M, ChatML, 2048 context, autoload enabled.
Best in Chrome or Edge because File System Access lets the ZIP stream directly to disk. Firefox/Safari use an in-memory fallback, which may not handle the full model comfortably.
