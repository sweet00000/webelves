The family

Four tools. One format.

Every tool produces or consumes the same portable .elf file. Start with the free ones — no account, nothing leaves your machine.

NotebookELF Free

Add sources — PDFs, URLs, paste — chat with grounded citations, then export a self-contained .elf. A three-panel studio (Sources · Chat · Studio) where every answer traces back to a specific passage.

Open NotebookELF →
Codelves Free

Real Python in a browser tab — NumPy, Pandas, Matplotlib, the whole stdlib via Pyodide. Files persist in OPFS. No server, no install.

Open Codelves →
Webelves Pro

Perplexity-style search, fully offline. Fetches pages, ranks by embedding similarity, and answers with local GGUF inference — all client-side. No API key, no queries transmitted.

Forge Ultra

An AI synthesis pipeline you can run. Give it topics — it writes, grades, refines, and packages knowledge articles as portable .elf files. Generate → judge → refine → package.
